# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please open an issue or email me directly. I'll respond within 48 hours.

Do not open public issues for critical vulnerabilities — use email instead.

## Scope

- This repository contains **data only** — JSON collections of USSD codes and JSON Schemas.
- No executable code runs on users' devices from this repo.
- Collection imports are validated against a JSON Schema and a CI validation script.

## Supported Versions

Only the latest version of each collection receives updates. Always point to the latest `main` branch when importing.
